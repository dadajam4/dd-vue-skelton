Typography

Typography

これはページのイントロダクションです。

Examples

heading

htmlの見出し系です

h1
Page Title ページタイトル
h2
Level-2 セクションの見出しです
h3
Level-3 セクション内の小見出しです
h4
Level-4 何かのタイトルです
h5
Level-5 何かのサブタイトルです
h6, caption
Level-6 キャプションです

body

コンテンツ本文系です

strong, em, b
This is Bold Body. 強調された本文です
b1
This is Normal Body. 標準の本文です

size

テキストサイズのバリエーションです

xl
This is 最大
lg
This is 大きい
sm, small
This is Small. スモールです
