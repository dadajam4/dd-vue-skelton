Tiles

dd-tile コンポーネントを利用する事で、データレコード等の情報に関連付けられた一連の項目をまとまりのあるレイアウトで表示できます。
このコンポーンントは単一のアバター要素、見出しを含むテキスト要素、及びいくつかのアクション要素で構成され、必要に応じて dd-tile-group でグルーピングする事が可能です。

link Avatar

dd-tile-avatar コンポーネントは、 dd-tile コンポーネント直下の先頭、もしくは末尾に設置してください。
本コンポーネントは dd-avatar のエイリアスとして動作します。従ってインターフェースの詳細はdd-avatarを参照してください。
複数のタイルをグルーピングする際、アバター表示の有無によらずレイアウトを統一するためには、空のアバター要素を利用します。

link Action

dd-tile-action コンポーネントもまた、 dd-tile コンポーネント直下の先頭、もしくは末尾に設置してください。
このコンポーネントは基本的に、1つのタイルに対するアクションを行う要素を挿入するために利用します。
本コンポーネント直下に要素を複数設置した場合、縦に並ぶレイアウトとなります。
アクション要素に対する補助的なテキストを表示する場合には、 dd-tile-action-text を使用してください。

link Content

dd-tile-content コンポーネントは、 dd-tile コンポーネント直下に必ず設置してください。
本コンポーネントは任意の dd-tile-title 、 dd-tile-sub-title 、 dd-tile-text で構成されます。
これらの要素はデフォルトでは改行が禁止され、エリアの横幅をテキストがはみ出る場合は省略表示されます。
この改行を許容させるには、 v-tile に対して wrap 属性を指定してください。

link Clickable

dd-tile は、以下の条件を満たす時にクリッカブルな要素となります。
  1. click リスナが設定されている時
  2. href 属性が設定されている時
  3. to 属性が設定されている時
つまり本要素は、メニュー要素やリンク要素にも利用可能です。
また従って、本要素は routable な要素です。

link Divider

dd-tile を単一の情報を表示するために利用する事はきっとめずらしいケースで、リストのようなn個のデータを表示する時に利用する事が殆どでしょう。
本コンポーネントに区切り線を表示させるためには divider 属性を指定してください。
この区切り線は時に、アバター要素の左スペースを覗いて表示させたいケースがあります。そのような場合には divider 属性に pull を指定してください。

link Grouping

dd-tile-group を利用する事で複数のタイルをグルーピングする事が可能です。
「グルーピングすると何がいいの？」ですが、たとえば各タイルの状態を dd-groupで一元的に設定可能です。具体的に設定できる項目は以下です。
  1. 区切り線（dd-divider） ※末尾の要素の区切り線は自動で非表示にします
  2. 無効状態（disabled）
  3. 改行許可（wrap）
  4. 狭め設定（dense）
