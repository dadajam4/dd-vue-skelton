Inputs

DVSではユーザーの入力を受け取るための便利なコンポーネントが多数用意されています。
これらは主にhtmlのフォーム部品で構成されていますが、中には通常のhtml要素には定義されていない便利なコンポーネントも含まれています。
Vueでアプリケーションを構築する際は、 v-model を利用してユーザー入力をハンドリングするのがベストプラクティスですが、 DVSではこれを強制しません。つまりv-modelを設定しない状態でもDOMツリー上には form タグで送信するための input 要素が常に存在し、 ユーザーの入力に応じてvalueがリアクティブに更新されています。
