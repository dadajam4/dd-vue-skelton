Multiple

この場合内部的には2つのinput要素が生成され、 v-model 利用時は配列が同期されます。

Tick mark

ticks 属性を指定する事で、操作中の間入力が可能な位置（step属性で指定された位置）に目盛り表示が行われます。
この表示を操作中ではなく常に表示させるには、 persistent-ticks 属性を指定してください。

Thumbnail label

thumb-label 属性を指定する事で、操作中の間サムネイルにラベルを表示可能です。
操作中のみでなく常にラベルを表示させる場合、 persistent-thumb-label 属性を指定してください。
thumb-label 属性にコールバックを指定する事で表示内容をカスタマイズ可能です。このコールバックはプレーンなテキスト、もしくはvnodeインスタンスを返却してください。
