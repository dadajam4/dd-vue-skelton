Selection controls

dd-checkbox 、 dd-radio 、 dd-switch コンポーネントを利用する事で、ユーザーはオプションを選択できます。
v-model と同時に利用する場合において、 dd-checkbox のみ、複数選択型である事から配列になります。これを他と同様単一選択とさせるには multiple 属性にfalseを指定してください。
これらのコンポーネントにvalueを指定しなかった場合、デフォルト値としてvmの_uidプロパティが設定されます。
またこれらのコンポーネントにおいてはデフォルトではインラインレイアウトで表示されます。ブロックレイアウトで表示させるためには、 inline 属性に false を指定します。

Checkbox indeterminate

チェックボックスのみ、 indeterminate 属性を設定する事で「不確定」な状態を示す表示が可能です。
これは例えば、いくつかのチェックボックスのグループを全選択、および全解除するようなアクションを行うのに利用できます。

Grouping & validation

ラジオ、チェックボックス、スイッチは基本的に単一で利用する事は少ないです。
これらの要素は dd-xxx-group でグルーピング可能です。
いくつかの属性はこの親コンポーネントから一括で注入可能です。
グルーピングする際はv-modelやバリデーションはこの親コンポーネントに設定してください。
