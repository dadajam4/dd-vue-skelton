Accordion

dd-accordion コンポーネントを利用する事で、いわゆる縦にぐにょーんと開閉するあの場所を設置可能です。
本要素は dd-accordion-title と、内包する$slots.default の2ブロックで構成されます。
複数のアコーディオンをグルーピングする際は dd-accordion-group を利用してください。

link Basic

dd-accordion-title を設置する事で、アコーディオンを開閉できます。
dd-accordion-title にはデフォルトで左側に開閉状態を示すアイコンが設置されます。
icon 属性を指定する事で、任意のアイコンを挿入でき、その場合デフォルトのアイコンは右側に移動します。
アイコンのスペースのみを設けるためには icon に空の情報を設定してください。

link Styled

styled 属性を設定する事で、いくつかの装飾が行われた状態で表示できます。
これは基本的に dd-accordion-title への任意の色設定と併用します。
この状態の時は、タイトル部分に配色が行われている事が前提である理由から、デフォルトのアイコンは常に右側に設置されます。
またボディ部分には余白が設けられます。

link Grouping

dd-accordion-group をを利用する事で複数の dd-accordion をグルーピングできます。
styled 属性を dd-accordion-group に設定した場合、子の dd-accordion に受け継がれます。
グループ内のアコーディオンはデフォルトではいずれか1つのみがアクティブとなるよう排他制御されます。
これをキャンセルするためには exclusive 属性に false を設定してください。
dd-accordion-group の中には dd-accordion 以外の要素も設置可能です。
