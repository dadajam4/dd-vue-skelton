Basic

dd-date-picker は locale 属性を注入する事で言語の設定が可能です。
未指定の場合、ブラウザがサポートしているデフォルトの言語が設定されます。

Menu and dialog

メニュー、またはダイアログで選択させる事ができます。
メニューの場合は dd-menu-fragment > dd-date-picker-menu
ダイアログの場合は dd-dialog-fragment > dd-date-picker-dialog の構造でマークアップしてください。
いずれの場合も cancellable 属性を設定する事でキャンセル、OKを選択させる事ができます。
この場合においてはメニューやダイアログエリアの外をクリックした時やESCキー押下においてもキャンセル扱いとなり、メニュー・ダイアログ表示開始時の値にロールバックされます。
