Defined

<dfn>

用語が使用されていることを示す

いま解説しているHTMLは、ウェブページを作成するための言語の一つです。

HTMLはそれほど難しいものではありません。簡単なウェブページなら数種類のタグを覚えれば十分に作成することが可能です。

<abbr>

略語や頭字語であることを表す

WHATWGは、2004年にHTML5の仕様策定を開始しています。

WHATWGは、当時W3Cが進めていたウェブ技術の標準化の方向性に満足しなかったApple、Mozilla、Operaによって設立されたグループです。

<time>

日付や時刻を正確に示す

ジョンとポールが初めて出会ったのは、よく晴れた夏の土曜日だった。ポールは、教会の夏祭りで歌うジョンをたまたま観て、歌がうまくてカッコいいと思ったそうだ。

私がビートルズを聴くようになったのは、中学生になった頃だろうか。そのころビートルズはすでに解散して伝説となっていた。

今夜は、23:00からラジオでビートルズ特集を放送するらしい。

<cite>

作品のタイトルを表す

夏目漱石の草枕は、こんな風に始まります。

山路を登りながら、こう考えた。
智に働けば角が立つ。情に棹させば流される。意地を通せば窮屈だ。とかくに人の世は住みにくい。

この作品の主人公は30歳の洋画家ですが、世間にあまりなじめていない彼の芸術家らしい生きざまがすでに冒頭で表現されています。

缶コーヒーがお買い得！

<q>

引用句・引用文であることを表す

夏目漱石の草枕の中に出てくる 智に働けば角が立つ。情に棹させば流される。意地を通せば窮屈だ。 という表現は大変有名です。

声や心の中で思ったことなど、他と区別したいテキストを表す

ジョバンニは思わずかけよって博士の前に立って、 ぼくはカムパネルラの行った方を知っていますぼくはカムパネルラといっしょに歩いていたのです と云おうとしましたがもうのどがつまって何とも云えませんでした。

<var>

変数であることを示す

鶴がx羽と亀がy匹います。頭の数はあわせて8で足の数はあわせて26です。xとyの値を求めなさい。

<samp>

プログラムによる出力結果のサンプルであることを示す

Styled

<sub>

下付き文字を表す

i番目の座標は、 (xi, yi) で表すことができます。例えば、１０番目の座標は (x10, y10) となります。

<sup>

上付き文字を表す

The most beautiful women are Mlle Gwendoline and Mme Denise.

（訳）もっとも美しい女性は、グウェンドライン嬢とデニス夫人です。
